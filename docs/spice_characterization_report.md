## Analysis

**LR Custom Cell:**
- Very high write margins at SS corner (66%) due to symmetric, large transistors
- Drops at FF/high-VDD (22-23%) — strong PMOS fights the write driver
- Uniform W=0.42 makes it easy to write but read stability is tighter
- 2.3x larger than foundry cell

**Foundry Cell:**
- Consistent write margins at FF corner (36-52%) thanks to asymmetric sizing
- Tightest at SS/low-voltage (12-14%) — the worst-case production corner
- HVT PMOS reduces leakage and improves read stability
- Doubled pull-down (cell ratio 2.0) strengthens read SNM
- 2.3x smaller, uses SRAM-specific design rules

**Both cells pass all 9 PVT corners at 27°C.** Temperature sweep (-40°C/125°C) not yet performed.

## Methodology

- **LR cell netlist:** Generated by `sky130_6t_lr.py`, standard SKY130 transistor models
- **Foundry cell netlist:** Adapted from OpenRAM `sky130_fd_bd_sram__openram_dp_cell` subcircuit (single-port configuration), uses SRAM-specific transistor models
- **Testbenches:** Generated by `rekolektion.verify.spice.generate_testbenches()`
- **Simulator:** ngspice with `set ngbehavior=hsa` for SKY130 model compatibility
- **Write margin:** DC sweep of BL from VDD to 0, measures trip point where Q crosses QB
- **Transient:** Precharge → write 1 → read 1 → write 0 → read 0 sequence
